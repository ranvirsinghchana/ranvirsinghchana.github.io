Bike-Share Usage Pattern Analysis

SQL | Python | Pandas

Project Overview

Processed 5.7M+ bike-sharing records using SQL and Python to analyze user behavior patterns across 600 docking stations, identifying key conversion opportunities.

  • Python and Pandas for data processing

  • Data visualization

Key Achievements
